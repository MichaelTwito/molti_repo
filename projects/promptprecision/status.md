# PromptPrecision — Status

This file is the **single source of truth** for the *current* PromptPrecision iteration.

## How to use
- Keep **only one** active iteration in this file.
- When you start a new iteration:
  1. Fill in the **Current iteration** fields below.
  2. Append a new section to `iteration_log.md` (append-only).
- When you finish an iteration:
  1. Set **Status** to **Done** (or **Stopped**).
  2. Append a short outcome + artifact links to `iteration_log.md`.
